## 项目需求

本项目是一个考试数据分析系统，主要功能包括：

1. **数据上传与解析**：支持上传Excel格式的考试数据文件，自动解析数据结构
2. **班级平均分计算**：计算各个班级的考试平均分，包括全卷、信息学科和通用学科
3. **学科分类分析**：将考试数据按信息学科和通用学科分类，分别进行详细分析
4. **自然排序**：按题目编号进行自然排序，确保分析结果的逻辑顺序
5. **可视化分析**：生成带色阶效果的Excel分析报告，直观展示成绩分布
6. **Web界面**：提供友好的Web界面，支持文件上传、结果展示和导出

## 系统功能说明

1. **文件上传**：支持上传 .xls 和 .xlsx 格式的Excel文件
2. **数据解析**：自动解析Excel文件结构，提取考试数据
3. **班级分析**：计算各个班级的平均分和学科成绩
4. **学科分析**：分别分析信息学科和通用学科的成绩分布
5. **结果展示**：在Web界面上展示分析结果，支持表格高亮和色阶效果
6. **导出功能**：生成Excel格式的分析报告，支持下载

## 使用说明

1. 启动Flask应用：
   ```bash
   python app.py
   ```

2. 打开浏览器访问：
   ```
   http://localhost:5000
   ```

3. 上传Excel格式的考试数据文件

4. 等待系统分析完成，查看分析结果

5. 可以选择导出Excel格式的分析报告

## 注意事项

1. 上传的Excel文件应包含正确的考试数据结构
2. 系统会自动处理表头和数据格式，但建议保持标准格式
3. 分析结果会保存在临时文件中，系统会定期清理过期文件
4. 对于大型Excel文件，分析过程可能需要较长时间
